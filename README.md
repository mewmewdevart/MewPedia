## Sobre
> O **MewPedia** é minha base pessoal de conhecimento, onde documento aprendizados, experimentos e conceitos importantes ao longo da minha jornada como desenvolvedora.
Este repositório funciona como um hub organizado de estudos, com o objetivo de consolidar conhecimento e facilitar consultas futuras, além de compartilhar conteúdo com a comunidade.

## Conteúdos
Aqui você vai encontrar anotações sobre:

* 💻 Front-end (HTML, CSS, JavaScript, React, etc.)
* ⚙️ Back-end (Node.js, APIs, bancos de dados)
* ♿ Acessibilidade e boas práticas
* 🎨 UI/UX e Design
* 🚀 Conceitos gerais de desenvolvimento

## Objetivo
* Organizar meus estudos de forma estruturada
* Consolidar aprendizados na prática
* Criar uma base de consulta rápida
* Compartilhar conhecimento com outras pessoas
